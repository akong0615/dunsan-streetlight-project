# dunsan-streetlight-project
dunsan streetlight project

대전 둔산동 지역에 설치된 가로등 위치를 지도 위에 시각화한 프로젝트입니다.

공공데이터로 제공되는 가로등 좌표(위도, 경도)를 활용하여  
가로등의 공간 분포를 한눈에 확인할 수 있도록 하였습니다.

---

## 프로젝트 내용

- 가로등 위치 데이터를 지도 위에 점(Point)으로 표시
- 낮 / 밤 배경 지도를 전환하여 확인 가능
- 가로등 분포를 시각적으로 비교

---

## 사용 데이터

- 대전 둔산동 가로등 설치 위치 (공공데이터)
- 위도, 경도 좌표 기반

---

## 실행 방법

1. 본 저장소를 클론합니다.
2. 제공된 HTML 파일을 웹 브라우저에서 실행합니다.
